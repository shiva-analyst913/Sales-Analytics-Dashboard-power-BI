## Sales Analytics Dashboard | Power BI Project
# Project Overview

This project is an interactive Sales Analytics Dashboard built using Microsoft Power BI. The dashboard provides comprehensive insights into sales performance, customer behavior, product performance, and operational efficiency. The project uses a sales dataset containing 2,800 records and 42 columns.

The data was cleaned and transformed using Power Query, and key business metrics were created using DAX. The dashboard is designed to help business users monitor KPIs, identify trends, and make data-driven decisions.

---

# Dataset Information

- Rows: 2,800
- Columns: 42
- Time Period: 2021–2023
- Tool Used: Microsoft Power BI
- Data Cleaning: Power Query
- Data Modeling & Calculations: DAX

---

# Data Cleaning & Transformation

- Removed duplicate records
- Handled missing (Null) values
- Replaced Unknown values with "Others" where applicable
- Corrected data types
- Fixed negative values where required
- Standardized category names
- Created calculated columns
- Improved data quality using Power Query
